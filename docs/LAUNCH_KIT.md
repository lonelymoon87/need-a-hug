## One-line pitch

Need a Hug is a lightweight emotional first-aid Agent Skill that teaches AI coding assistants to comfort overwhelmed users before giving them another plan.

## Short pitch

Most AI agent skills make assistants faster, sharper, or more autonomous. `need-a-hug` does something different: it helps an agent notice when the person using it is tired, ashamed, frustrated, burned out, or close to giving up.

When triggered with `need a hug`, `/hug`, `comfort me`, or a clear distress signal, the agent slows down, comforts first, avoids turning pain into an immediate checklist, and returns to the task in smaller steps only when the user is ready.

It is not therapy, diagnosis, medical care, or emergency support. It is a small, text-only emotional first-aid layer for AI tools.

## Who it is for

- Developers who use AI coding agents every day.
- People who get overwhelmed when agents keep pushing tasks forward.
- Agent builders who want a safer person-first fallback for distress moments.
- Claude Code, Codex, Cursor, Kiro, VS Code, OpenCode, OpenClaw, Antigravity, CodeBuddy, and GitHub Copilot users experimenting with skill systems.

Most skills teach agents to move faster, search deeper, or automate more. This one does the opposite: when the user is overwhelmed, ashamed, burned out, frustrated with the agent, or explicitly asks for comfort, it tells the agent to slow down, comfort first, and only return to the task in smaller steps when the user is ready.

It is text-only, MIT licensed, and intentionally not therapy or medical care. It is more like an emotional first-aid layer for AI coding tools.

It ships as a portable `.skill` file and as a Claude Code plugin, with adapters for Codex, Cursor, Kiro, VS Code, OpenCode, OpenClaw, Antigravity, CodeBuddy, and GitHub Copilot-style instruction surfaces.
